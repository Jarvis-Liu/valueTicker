# Agent Behavior Guidelines

## Core Constraints
1. **Mandatory Documentation**: You must write comprehensive function comments for EVERY function or method you create or modify. Do not skip this under any circumstances.
2. **Format**: Use the standard documentation format for the target language (e.g., JSDoc for JavaScript/TypeScript, Docstrings for Python).

## Definition of Done (DoD)
A task is NOT considered complete unless all accompanying functions have explicit comments defining parameters, return values, and behavior.
